# Vigenère Cipher Decrypter (CTF Challenge)

A lightweight PHP script designed to solve a Vigenère cipher cryptography challenge using a repeating key. It compares standard key alignment behavior against non-skipping indexing methods when encountering special characters like brackets and underscores.

## Challenge Details
* **Ciphertext:** `GXWVUU{ZMKZXXTX_LV_YCBZXKNA}`
* **Key:** `CTF`
* **Rule:** Subtract the corresponding key letter cyclically ($Cipher - Key = Plaintext$) and skip non-alphabetic characters.

## The Right Answer
Based on the cryptographic hint, the correct flag is generated by **skipping special characters** (`$flag1` in the script):

```text
EERTBP{XTFXESRE_GT_FXZGSIUV}
```

### Mathematical Proof
Applying the reverse Caesar shift ($Letter - Key$) dynamically:
* `G` (6) - `C` (2) = **E** (4)
* `X` (23) - `T` (19) = **E** (4)
* `W` (22) - `F` (5) = **R** (17)
* `V` (21) - `C` (2) = **T** (19)
* *The `{` bracket is skipped, keeping the key pointer aligned perfectly for the next letter.*

## How the Code Works
The script runs a single loop over the ciphertext but tests two distinct string outputs:

1. **`$flag` (Without Skipping):** Increments the key pointer globally on every character. This yields `EERTBP{GHIGSVAS_SQ_FXZGSIUV}` (Incorrect).
2. **`$flag1` (Skipping Special Characters):** Freezes the key tracking variable (`$ki`) when encountering `{`, `_`, or `}`. This yields the correct solution.

## Execution
Run the script using any local PHP environment or via the command line:
```bash
php solution.php
```
